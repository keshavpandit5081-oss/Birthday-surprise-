# 🎂 Happy 23rd Birthday Ekku ❤️

A beautiful birthday surprise webpage created with love for Ekku's 23rd birthday.

## Features

✨ **Interactive Celebration**
- Confetti animation with a click of a button
- Beautiful pink-themed design
- Responsive layout for all devices

📸 **Photo Gallery**
- Display up to 10 memorable photos
- Auto-responsive grid layout
- Hover animations

🎥 **Video Section**
- Embedded video player
- Full video controls

💌 **Love Letter**
- Heartfelt message
- Personalized birthday wishes

## File Structure

```
├── index.html       # Main webpage
├── style.css        # Styling and animations
├── script.js        # Interactive features
├── photos/          # Photo gallery (add your photos here)
│   ├── photo1.jpg
│   ├── photo2.jpg
│   └── ...
└── videos/          # Videos section (add your videos here)
    ├── video1.mp4
    └── video2.mp4
```

## How to Use

1. Add your photos to the `photos/` folder
2. Add your videos to the `videos/` folder
3. Share the link with Ekku and click the button to celebrate! 🎉
